# Project Context

This project is a Flutter mobile application developed for the animal rescue NGO "Patinhas e Amor".

The application will be used internally by the NGO to manage animal rescue cases reported by the public through a web platform.

Reports include cases such as:

* abandoned animals
* animal abuse
* injured animals

These reports are stored in a database and accessed through a REST API.

The Flutter app allows the NGO team to:

* view reported occurrences
* update the status of occurrences
* register rescued animals
* manage rescue information

This project is part of a university extension initiative aimed at supporting real community organizations using technology.
