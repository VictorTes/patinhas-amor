# UI Screens

The application should include the following screens.

## 1. Login Screen (optional)

Simple login for NGO members.

Fields:

* email
* password

---

## 2. Home Screen

Displays a list of occurrences.

Each card should show:

* occurrence type
* location
* status
* date

Clicking a card opens the details screen.

---

## 3. Occurrence Details Screen

Displays:

* full description
* location
* image (if available)
* contact information

Actions:

* update status
* mark as resolved

---

## 4. Register Animal Screen

Form for registering a rescued animal.

Fields:

* name
* species
* age
* description
* upload image
* rescue date

---

## 5. Animals List Screen

Displays all animals registered by the NGO.
