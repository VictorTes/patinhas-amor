# Product Requirements

## Main Features

### 1. View Occurrences

The app must display a list of reports submitted by the public.

Each occurrence contains:

* type
* description
* location
* date
* status

Possible status values:

* pending
* in_progress
* resolved

---

### 2. Occurrence Details

The user should be able to open an occurrence and see all details.

The user must also be able to update the occurrence status.

---

### 3. Register Rescued Animal

After responding to an occurrence, the NGO can register the rescued animal.

Fields required:

* name
* species
* approximate age
* description
* image
* rescue date
* status

Animal status values:

* under_treatment
* available_for_adoption
* adopted

---

### 4. Dashboard (optional)

Simple statistics:

* number of pending occurrences
* animals under treatment
* animals available for adoption
