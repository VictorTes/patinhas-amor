# Technical Specification

## Framework

Flutter

## Language

Dart

## Architecture Pattern

Simple layered architecture:

UI → Services → Models

## Folder Structure

```text
lib/
  main.dart
  screens/
  widgets/
  models/
  services/
  repositories/
  utils/
```

## State Management

Use Provider or Riverpod.

## Networking

Use HTTP package to communicate with the REST API.

## Data Format

JSON
