# Development Tasks

## Phase 1 - Project Setup

* Initialize a Flutter project
* Configure folder structure
* Add dependencies:

  * http
  * provider or riverpod
  * image_picker

---

## Phase 2 - Models

Create Dart models for:

* Occurrence
* Animal

Each model should support JSON serialization.

---

## Phase 3 - API Services

Create service classes:

OccurrenceService
AnimalService

Responsibilities:

* fetch occurrences
* update occurrence status
* create animals
* fetch animals

---

## Phase 4 - UI

Implement the following screens:

1. Home screen (occurrence list)
2. Occurrence details
3. Register animal form
4. Animals list

---

## Phase 5 - Polish

* loading indicators
* error handling
* clean UI layout
